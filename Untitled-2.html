Introduction and Objectives

This project analyzes the King County housing market using machine learning to study the key drivers and trends impacting the price of houses. The study utilizes clustering KMeans and Agglomerative in addition to time forecasting using ARIMA to profile similar houses and predict the price changes over time.

Motivation

The motivation stems from the necessity of acquiring actionable insights in real estate. The complexity of the location, size and quality relationship, and other key determinants of value is pronounced, and machine learning can increases the value of prediction and strategic decision making.

Data Preparation and Feature Engineering

Data was cleaned to remove missing or unrealistic values. Two derived attributes were added: house_age (age at sale) and price_per_sqft (value per square foot). Outliers were trimmed to improve clustering stability.

In[20]:
df['date'] = pd.to_datetime(df['date'], errors='coerce')

# Useful derived attributes
df['year'] = df['date'].dt.year
df['month'] = df['date'].dt.month

# Approximate age at the time of sale
df['house_age'] = df['year'] - df['yr_built']

# Price per square foot
df['price_per_sqft'] = df['price'] / df['sqft_living']

# Remove records with unrealistic or critical null values
cols_core = ['price','sqft_living','lat','long','zipcode','year','month','house_age','price_per_sqft']
df_clean = df.dropna(subset=cols_core).copy()

# Trim extreme outliers of price_per_sqft (1%99%) for more stable clustering
low, high = df_clean['price_per_sqft'].quantile([0.01, 0.99])
df_clean = df_clean[(df_clean['price_per_sqft']>=low) & (df_clean['price_per_sqft']<=high)]

Clustering

KMeans was tested for K=210. The optimal K was chosen based on maximum Silhouette and minimum DaviesBouldin Index. Agglomerative clustering was also applied for comparison.

In[26]:
# Evaluate KMeans for K=2..10
results = []

import itertools
import statsmodels.api as sm

def fit_aic(train_series, pdq):
    try:
        model = sm.tsa.ARIMA(train_series, order=pdq)
        res = model.fit()
        return res.aic, res
    except:
        return np.inf, None

pdq_grid = list(itertools.product(range(0,4), range(0,3), range(0,4)))
scores = []
for pdq in pdq_grid:
    aic, res = fit_aic(train, pdq)
    scores.append((pdq, aic))

scores_sorted = sorted(scores, key=lambda x: x[1])
